## Read in some data

absolute directions- go from the very first folder on your hardrive.
relative path (use this) - start from where you are now. it assumes that
the file you are trying to get to is close to where your code file is.
These are stable because the relative paths can be used by others
reading the code. absolute paths only apply to your own computer

benefit of using an R project is you can start construction relative
paths from there.
